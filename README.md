# Madam Ji Photo Request Website

A simple static website made for a sweet, respectful photo request.

## Files

- `index.html` - welcome page
- `ask.html` - the main request
- `reasons.html` - cute reasons page
- `reply.html` - final WhatsApp reply page
- `styles.css` - all styling and illustration
- `script.js` - the tiny interactive "I'll see" button

## Hosting

You can host this folder anywhere static sites are supported:

- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages

No build step is needed. Upload the files in this folder and set `index.html` as the entry page.

The site is plain HTML, CSS, and JavaScript, so it works across Windows, iOS, Android, and modern desktop/mobile browsers.
